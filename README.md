# Flowtype
Flowtype is a minimalist code typing practice application built with Vue.js and Tailwind CSS. It helps developers improve muscle memory using dynamic code snippets generated by Google Gemini 1.5 Flash AI.

### Key Features
* **AI-Powered Snippets:** Dynamic code generation with static fallback mechanisms.
* **Performance Metrics:** Real-time WPM, Accuracy, and Error tracking.
* **Modern UI:** Clean, responsive, and developer-friendly design.

### Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Add your `VITE_GEMINI_API_KEY` to `.env.local`.
4. Launch development server: `npm run dev`.
